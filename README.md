<!-- GETTING STARTED -->
## Pokud se vám ale video nechce sledovat zde jsou jednotlivé kroky!



### Návod
1. Stákni si VSC link: [https://code.visualstudio.com/Download]
### Installation

2. Stáhni si Node.Js link: https://nodejs.org/en/

3. Vytvoř si novou aplikaci na: https://discord.com/developers/applications
 
4. Zkopíruj si kód
```sh
   git clone https://github.com/exotwd/tutorials
   ```
5. Stáhni si npm packages
   ```sh
   npm i
   ```
6. Vlož token svého bota do `config.js`
   ```JS
   "token": "tvůj token",
   ```
7. Uprav prefix v `config.js`
  ```JS
  "prefix": "tvůj prefix",
  ```
  
8. Otevři terminál a napiš 

```sh
   node . 
   ```
   
   Bot by měl být onlineuž ho stačí jen pozvat na servery kde má být!
   
   (jak na to se dozvíš ve videu!)
